# AI学习课程资源汇总
整理了6门官方优质AI与计算机基础课程，全部免费学习

---

## 1. 吴恩达《AI For Everyone》
**官方地址**：
- [DeepLearning.AI官网](https://www.deeplearning.ai/courses/ai-for-everyone/)
- [Coursera学习页](https://www.coursera.org/learn/ai-for-everyone)

**介绍**：全球最经典的AI零基础入门课，完全不需要编程基础。吴恩达亲自讲授，讲解AI的基本概念、AI在商业中的落地方法、AI对社会的影响，是所有想了解AI的人的首选入门课。

---

## 2. OpenAI Prompt Engineering 学习资料
**官方学习资源**：
- [OpenAI官方提示词工程指南](https://platform.openai.com/docs/guides/prompt-engineering)
- [吴恩达×OpenAI 开发者提示词课程](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/)

**介绍**：最权威的提示词工程学习资料：官方指南包含6大核心提示词底层策略，彻底告别"背咒语"；合作课程由OpenAI提示词工程师和吴恩达共同讲授，教授如何基于OpenAI API构建LLM应用。

---

## 3. Google《Generative AI》学习路径
**官方学习地址**：
- [Google Cloud生成式AI完整学习路径](https://cloud.google.com/training/generative-ai-learning-path)
- [Google AI Essentials 零基础入门课](https://grow.google/certificates/en_us/certificates/ai)

**介绍**：Google官方推出的完整生成式AI学习体系，包含10门独立课程，从生成式AI基础原理、大语言模型、提示词工程、负责任AI，到Google Cloud上的AI应用开发与部署。

---

## 4. 哈佛 CS50Dx（CS50x）
**官方学习地址**：
- [CS50官网](https://cs50.harvard.edu/x/)
- [edX官方学习页](https://www.edx.org/course/introduction-computer-science-harvardx-cs50x)

**介绍**：全球最知名的计算机科学入门课，哈佛David J. Malan教授讲授，完全不需要编程基础。讲解算法思维、数据结构、C语言、Python、SQL、Web开发等计算机科学核心内容，是学习AI之前必备的计算机基础课程。

---

## 5. 哈佛 CS50P
**官方学习地址**：
- [CS50官网](https://cs50.harvard.edu/python/)
- [edX官方学习页](https://www.edx.org/learn/python/harvard-university-cs50-s-introduction-to-programming-with-python)

**介绍**：CS50系列的Python专项编程课，专门针对AI开发必备的Python语言，不需要任何编程基础。讲解函数、变量、条件循环、异常处理、文件操作、正则表达式、面向对象编程等Python核心内容。

---

## 6. 微软《Generative AI入门》
**官方学习地址**：
- [微软开源完整课程](https://microsoft.github.io/generative-ai-for-beginners/)
- [微软Learn中文学习路径](https://learn.microsoft.com/zh-cn/training/paths/ai-fluency/)

**介绍**：全球最火的生成式AI实战入门课，GitHub星标超10万，共21节完整课程。从大模型原理、提示词工程、RAG检索增强、AI Agent开发、图像生成，到负责任AI与安全，每节课都配有Python和TypeScript双语言实战代码示例。
